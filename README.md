# Interview - 教师访谈记录

## 项目简介

本仓库用于结构化保存教师访谈记录，旨在了解教师在教学环节中的痛点与需求，为 EduTrack 产品设计提供真实用户洞察。

## 目录结构

```
interview/
├── README.md                    # 项目说明
├── transcripts/                 # 原始访谈记录（结构化整理）
│   ├── 01-环境科学AP老师.md
│   ├── 02-化学老师.md
│   ├── 03-语文老师.md
│   └── 04-综合访谈-生物数学教务.md
├── analysis/                    # 访谈分析
│   ├── pain-points-summary.md
│   └── 2026-03-03-综合访谈分析.md
└── insights/                    # 洞察与产品方向
    ├── product-opportunities.md
    └── 2026-03-03-产品机会与路线建议.md
```

## 访谈对象

| 编号 | 学科/角色 | 年级 | 关键痛点 |
|------|-----------|------|----------|
| 01 | AP 环境科学 | 高中 | 出题困难、项目化教学设计、Worksheet 制作 |
| 02 | 化学（AP + 高一） | 高一/高二/高三 | 学案制作、找题、判作业/登分、项目化学习 |
| 03 | 语文 | 高中 | 资源整合耗时、教案制作、公众号编辑 |
| 04 | 生物/AP + 数学 + 年级教务 | 高中 | 多教材整合、真题分类、批改反馈、教务协同割裂 |

## 访谈目标

帮助教师缓解教学压力，提升教学效率——聚焦费时耗力、重复性高的环节。
