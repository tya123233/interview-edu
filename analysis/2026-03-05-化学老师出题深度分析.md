# 访谈分析：化学老师（高一国际部）出题深度需求

> 基于 06 号访谈记录，2026-03-05

---

## 核心发现

### 1. AI 出题的 4 个具体失败点

这是目前所有访谈中**对 AI 出题缺陷最具体的反馈**：

| 缺陷 | 严重程度 | 产品启示 |
|------|---------|---------|
| 只能出概念题 | ████ 致命 | 需要支持"应用题""分析题""综合题"等多种认知层级 |
| 无难度梯度 | ████ 致命 | 需要布鲁姆分类法（记忆→理解→应用→分析→评估→创造）作为底层框架 |
| 输出混乱 | ███ 严重 | 需要结构化模板（题干、选项、分值、评分标准） |
| 无法呈现化学内容 | ████ 致命 | 需要 LaTeX/化学式渲染 + 图表生成能力 |

### 2. "好题"的设计框架（来自老师的教育学视角）

老师明确描述了一道好的考试题的构成：

```
好题 = 真实背景 + 知识嵌入 + 难度梯度 + 能力分层

例：
  背景：某工厂排放废水中含有重金属离子（真实场景）
  第 1 问：写出相关化学方程式（记忆/理解）
  第 2 问：计算所需药剂量（应用）
  第 3 问：分析处理效率随 pH 变化的趋势（分析）
  第 4 问：设计一个改进方案并说明理由（评估/创造）
```

**产品意义**：出题引擎的 prompt 应该按这个框架设计，而非简单的"出 5 道关于 X 的题"。

### 3. 老师真正需要的是"创意合作者"

> 老师把 AI 当"头脑风暴伙伴"用——不是让它出成品题，而是让它提供背景和创意。

当前使用模式：
```
老师：我想考氧化还原，有什么真实背景？
AI：电池、铁锈、漂白、食品保鲜...
老师：食品保鲜不错，帮我围绕这个出几道
AI：（出一些基础题）
老师：太简单了，第 3 问加一个图表分析
AI：（做不好）
老师：算了，图表我自己用 Desmos 画
```

**产品应匹配这个工作流**，而非试图替代整个过程。

### 4. 练习/测验/考试三级场景

不同场景的出题需求完全不同：

| 场景 | 频率 | 题目特点 | AI 辅助可行性 |
|------|------|---------|-------------|
| 课堂练习 | 每天 | 小、碎、单知识点 | ✅ 高（现有 AI 基本能做） |
| 章节测验 | 每章 | 适度综合，2-3 知识点交叉 | ⚠️ 中（需要知识图谱支撑） |
| 期末考试 | 每学期 | 高度综合，真实背景，梯度设计 | ❌ 低（目前 AI 做不到） |

## 与其他访谈的交叉验证

### 确认的共性痛点

| 痛点 | 提及老师 | 本次新增验证 |
|------|---------|------------|
| 非 AP 课程题库匮乏 | 02 化学、05 数学、**06 化学** | ✅ 高一国际部是空白区 |
| AI 出题质量差 | 01 环科、05 数学、**06 化学** | ✅ 最详细的缺陷描述 |
| 纸质作业 → 批改 → 登分繁琐 | 02 化学、**06 化学** | ✅ 但本次老师有教育理念支撑纸质选择 |
| 教师间无共享资源库 | **06 化学**（新发现） | 3 位化学老师各自独立 |
| 学生信息系统割裂 | 04 综合访谈、**06 化学** | ✅ ManageBac 看不到活动/竞赛 |

### 新发现的痛点

| 新痛点 | 产品方向 |
|--------|---------|
| 化学方程式/图表是 AI 的硬性技术瓶颈 | 需要 LaTeX + 化学式渲染 + 图表生成 |
| 考试题的教育学设计框架（背景+梯度） | 出题引擎需内置布鲁姆分类法 |
| 老师用 AI 当"头脑风暴伙伴" | 产品定位应是"创意合作者" |
| project 数据库缺少学生元信息 | 项目管理需关联学生档案 |
| 不同题目类型（练习/测验/考试）需求不同 | 出题引擎需要场景选择 |

## 产品建议

### 优先级排序（基于本次访谈）

1. **P0 — 分场景出题引擎**
   - 支持练习/测验/考试三种场景
   - 内置布鲁姆分类法做难度梯度
   - 支持"真实背景 + 知识嵌入"的出题框架
   - 化学方程式渲染（LaTeX/mhchem）

2. **P0 — 创意合作模式**
   - 不是一次性出成品题
   - 而是对话式逐步打磨：背景→知识点→题目结构→细节
   - 匹配老师"反复磨"的实际工作流

3. **P1 — 化学学科专项支持**
   - 方程式输入/输出
   - 图表生成（结合 Desmos API？）
   - 分子结构可视化

4. **P2 — 教师素材库**
   - 个人题库管理
   - 往年试卷存档和检索
   - 教师间可选共享
